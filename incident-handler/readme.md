## Application Setup

* ### Setup
    1. Clone the codebase
    2. Import the project as maven project
    3. Execute 'mvn clean install'
    4. mvn spring-boot:run
    
## Commands:
  ### Install

```sh
mvn clean install
```

  ### Run
  
```sh
mvn spring-boot:run
```
